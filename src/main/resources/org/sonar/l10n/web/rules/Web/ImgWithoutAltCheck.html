Visually impaired readers rely on screen reader softwares to render web pages using speech. Images are rendered using the alternate text attribute alt. Non-graphical browsers or browsers for which images are disabled also rely on this attribute. To improve accessibility, the alt attribute should therefore always be used.

The following code:

<img src="foo.png" />

should be refactored into:

<img src="foo.png" alt="Some textual description of foo.png" />
