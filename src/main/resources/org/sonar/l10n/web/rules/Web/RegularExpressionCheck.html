This rule checks elements starting tags against a given regular expression, and creates an issue with a given message whenever a match is found.

For example, expression can be set to (?i)<br\s*+> to find unclosed line break tags:

<br>        <!-- Non-Compliant -->
<Br>        <!-- Non-Compliant -->
<bR>        <!-- Non-Compliant -->
<br     >   <!-- Non-Compliant -->
<br />      <!-- Compliant -->
