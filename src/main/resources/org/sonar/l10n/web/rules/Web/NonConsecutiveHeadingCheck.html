Skipping heading levels is most of the time done for a bad reason: Benefit from that tag's CSS style. Instead, it is better in such case to tune the CSS rather than skipping levels.

The following code:

<h2>My Title</h2>     <!-- Non-Compliant - h1 is not being used -->

<h3>My Sub Title</h3> <!-- Compliant -->

should be refactored into:

<h1>My Title</h1>     <!-- Compliant -->

<h2>My Sub Title</h2> <!-- Compliant -->
